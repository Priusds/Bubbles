ParGeo
======

Generate complex two-dimensional geometries for simulations based on
mesh discretizations.

Installation
------------

ParGeo requires:

-  Python (version 3.11 or higher)
-  Gmsh (version 4.11.1 or higher)

You can install ParGeo from PyPI using pip:

.. code:: bash

   pip install pargeo
